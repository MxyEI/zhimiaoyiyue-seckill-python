#### 介绍
知苗易约小程序获取signature+主抢购程序，2022-11-28知苗易约小程序最新95版本

功能:
* 1.获取signature
* 2.获取pid和mxid
* 3.自动过验证码

### 获取signature

最新版知苗易约获取signature的办法只能hook底层微信，
市面上获取signature的方法
```
1.hook pc微信
2.CE软件查pc微信内存
3.hook ios微信
4.hook 安卓WX
```

本项目采用的是首款hook ios做的动态库插件，可获取任意小程序code和signature，非越狱手机也能使用
